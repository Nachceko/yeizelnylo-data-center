## 所需条件：

1. 一台可以正常运行的电脑。
2. 会使用浏览器。

## ==第一步==：安装Java

MCJava所有版本对应的Java版本如下：（AI生成）<br>

![](/mcjava-windows-install/01.png)
:::tip
请参考上方图片，根据你想玩的Minecraft版本选择对应的Java版本进行安装。<br>
==我这里已Java17为例进行说明。==
:::

点击Java17官网下载链接:[https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html](https://www.oracle.com/java/technologies/javase/jdk17-archive-downloads.html)
