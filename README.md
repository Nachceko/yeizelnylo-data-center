# yeizel-nylo-data-center

## YeizelNylo 数据中心

网站使用 [vuepress](https://vuepress.vuejs.org/) 和 [vuepress-theme-plume](https://github.com/pengzhanbo/vuepress-theme-plume) 构建生成。

## 安装

```sh
pnpm install
```

## 运行

```sh
# 单独生成 changelog
pnpm changelog
# 单独生成Tags
pnpm generate:tags
# 启动开发服务带有热更新
pnpm docs:dev:watch
# 单独启动开发服务
pnpm docs:dev
# 构建生产包
pnpm docs:build
# 本地预览生产服务
pnpm docs:preview
# 更新 vuepress 和主题
pnpm vp-update
```
